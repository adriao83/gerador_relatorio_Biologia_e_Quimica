# Gerador de Relatório de Estágio - UFAM (Biologia e Química)

Este script automatiza a criação de relatórios para o **Estágio Supervisionado no Ensino de Biologia** da UFAM, seguindo as normas da Licenciatura em Ciências.

## 📋 Funcionalidades
* Estrutura completa de 135 horas divididas em 6 etapas.
* Formatação automática em Word (ABNT/UFAM).
* Integração com IA para ajuste de textos acadêmicos.
* Extração automática de referências bibliográficas.

## 🚀 Como usar
1. Instale as bibliotecas necessárias:
   `pip install requests python-docx`
2. Rode o script:
   `python gerador_relatorio_ufam.py`

## 🛠️ Tecnologias
* Python 3
* Google Gemini API (Motor de IA)
* Python-docx (Gerador de documentos)
